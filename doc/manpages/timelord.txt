NAME
     timelord — Macintosh time server daemon

SYNOPSIS
     timelord [-d] [-l] [-n nbpname]

DESCRIPTION
     timelord is a simple time server for Macintosh computers that use the tardis client.

OPTIONS
     -d       Debug mode, i.e. don´t disassociate from controlling TTY.

     -l       Use the time zone sensitive localtime of the server. The default behavior when not using this option is GMT.

     -n nbpname
              Register this server as nbpname.  This defaults to the hostname.
