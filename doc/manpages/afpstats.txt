NAME
       afpstats - List AFP statistics

SYNOPSIS
       afpstats

DESCRIPTION
       afpstats list AFP statistics via D-Bus IPC.

NOTE
       afpd must support D-Bus. Check it by "afpd -V".

       "afpstats = yes" must be set in @pkgconfdir@/afp.conf.

SEE ALSO
       afpd(8), afp.conf(5), dbus-daemon(1)
